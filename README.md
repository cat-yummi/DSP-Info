# 总纲

当你读到本文件夹中的说明时，阅读者可能是人类，也可能是 AI；**大概率**正在执行与《戴森球计划》相关的任务。

《戴森球计划》是由柚子猫工作室 自主研发的一款大宇宙尺度的自动化沙盒游戏。

《戴森球计划》在互联网上的公开内容**很少**，且**多数不可靠**。除本仓库整理材料外，目前可优先采信的线上信源仅有：

- [戴森球计划 BWIKI（哔哩哔哩游戏 Wiki）](https://wiki.biligame.com/dsp/)
- [戴森球计划知识库 DSP-KB](https://dsp-kb.top/)

若你是 **AI**：当读到上述说明，或**上下文已被压缩**、对玩家用语与游戏实体不确信时，**强烈建议先阅读**本文件夹中的《名词解释》正文，避免与协作者之间出现指代、术语或机制上的偏差。

**不必**为完成任务而预先通读本文件夹下其余文件；**建议**先浏览各**文件名**，在需要查具体数据或序号时，再打开对应文件。

若你是**人类**、却希望与你协作的 AI 真正听懂你在《戴森球计划》语境里说的话，请考虑拉取本仓库。

- **不在主仓库里维护、只想拉名词资料**：在任意目录执行 `git clone https://github.com/cat-yummi/DSP-Info.git`。
- **在主仓库里维护、要首次挂载本子模块**：在仓库根目录执行 `git submodule add https://github.com/cat-yummi/DSP-Info.git 名词资料`。
