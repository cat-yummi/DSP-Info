# 单球距离换算（建造网格与米）

赤道一带建造网格每格约 1.257 米。

高度方向每格约 1.333 米。

因非赤道、非经度方向上的格长各不相同，严谨讨论距离时一般不说笼统的「格」，
而多用「米」，或明确说成「赤道格」等可对照的说法。

本文中的**格**，指《戴森球计划》在行星等地表上使用的**建造网格**（离散铺地单元），是游戏内正式机制，**不是**玩家口头随便说的长度词。同一词在蓝图讨论里有时还指**建筑占格**（传送带、分拣器接线范围等），与下文**地表建造网格**不是同一套概念。

官方对岩质行星表面网格、天体尺度等的归纳，见 BWIKI **[天体](https://wiki.biligame.com/dsp/%E5%A4%A9%E4%BD%93)**、**[岩质行星](https://wiki.biligame.com/dsp/%E5%B2%A9%E8%B4%A8%E8%A1%8C%E6%98%9F)** 等条目（以下分层表与常数摘自该处表述，游戏版本若有更新以 Wiki 为准）。

---

## 1. 建造网格和米不是一回事

- **米**：游戏世界中的长度单位；在球面上沿大圆、在空间中沿直线测量时，含义清晰。
- **格（建造网格）**：引擎为铺展建筑而划分的**地表单元**，是**离散编号**；**一格往往不等一米**，且随**纬度带、方向（东西 / 南北 / 径向）**变化，不能把「1 格 = 多少米」当成全行星统一的常数。

---

## 2. 为什么是「球 + 网格」：畸变与分界

行星表面是**球面**，建造网格在球面上铺开后，**同一网格单元对应的实际弧长会随纬度变化**；在不同纬度带之间还存在**网格分界**。在分界线上放建筑时，会**吸附到更靠近赤道一侧**的网格，即**分界线归更低纬度的纬度带**一侧（见 Wiki「岩质行星」等条）。

---

## 3. 岩质行星表面网格（官方分层）

**所有岩质行星的表面网格相同。** 从两极向赤道，网格按纬度带分层，各带用 **（经线格数 × 纬线格数）** 描述，大致如下（与 BWIKI 一致；南北半球对称处用「×2」表示各一半球上的格数规模）：

| 区域 | 分层（经线格 × 纬线格数） |
| --- | --- |
| 极地 | 4300 格 ×2（南北） |
| | 5×20（含极点，极点为一直径 9 的圆） |
| | 5×40 |
| | 5×80 |
| | 5×100 |
| | 10×160 |
| | 10×200 |
| 高纬度 | 78000 格 ×2 |
| | 15×300 |
| | 15×400 |
| | 25×500 |
| | 25×600 |
| | 50×800 |
| 低纬度 | 161000 格 |
| | 80×1000 |
| 赤道 | 1×1000（赤道带） |

- **全地表合计约 325600 格**；用地基完全铺满可完成成就 **「死星计划」**（名称与条件以当前版本游戏内为准）。

由此可见：地表网格是**多段、按纬度带划分的**，**不能**再用「整球经度一圈 1000、纬度半圈 500」这种**单一均匀网**去概括；只有在具体带（例如赤道 **1×1000**）上，才适合单独做「这一带每格约多少米」的换算。

---

## 4. 天体尺度是固定的（同类一致）

游戏内**不是**「所有星球都一样大」，
要先分清**岩质行星**与**巨行星**等类别。

- **岩质行星**：半径为固定规格（开发日志约定直径 **1D = 400 m**，半径 **200 m**）；
  任意两颗岩质行星，**球体大小相同**。
- **巨行星**（气态巨行星）：直径为固定规格（**4D = 1600 m**）；
  任意两颗巨行星，**球体大小也相同**；
  与岩质行星**不是同一尺寸**。

恒星、红巨星、天文单位等另有固定写法，
见文末附录表；与「每颗星球都一样」不是一回事。

---

## 5. 赤道一带与米的换算

赤道一带建造网格每格约 1.257 米。

开发日志与 Wiki 常用约定：行星直径 1D = 400 m（半径 200 m）。
仅针对赤道带「绕行星一周约 1000 格」时，
赤道大圆周长约 1257 m，与上句一致。
其它纬度带因分层不同，不得把该比例套到全行星。

---

## 6. 高度（径向）与米的换算

高度方向每格约 1.333 米。

垂直方向（堆叠、浮空等）使用另一套离散刻度，
与地表经线格、纬线格不是同一套网格。
常见约定：球半径 200 m 对应半径 150 格（直径常记 300 格），
即 200 ÷ 150，与上句一致。
与赤道一带约 1.257 m/格数值不同；
建造网格与米不能混成一个固定换算率。

---

## 7. 与蓝图「建筑占格」的提醒

蓝图工具、偏移与分拣器距离里常说的「几格」「约 1.8 m」等，多指**建筑碰撞与占格**，**不要**与 BWIKI 所载**地表 325600 建造网格**混为一谈；需要时分开讨论。
